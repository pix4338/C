# 쇼핑몰
## 1. 주요 기능
- 로그인 / 로그아웃
- 회원가입
- 정보 수정 / 조회
- 상품 조회
- 상품 관리 (추가/수정/삭제)
- 순이익이 많은 제품, 재고가 적은 제품 조회
